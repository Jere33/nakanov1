## Clone this project

```bash
> git clone https://github.com/Jere33/nakanov1
```

## Install the dependencies:
Before running the below command, make sure you're in the project directory that
you've just cloned!!

```bash
> cd nakanov1
> bash install.sh
```

### Usage
```bash
> npm start
```

### Setting
Change admin and all in folder

[`HERE`](https://github.com/Jere33/nakanov1/blob/main/admin/set.json)
